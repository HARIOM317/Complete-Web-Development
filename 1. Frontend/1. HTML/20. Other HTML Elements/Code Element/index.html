This is an example of how to display a code snippet using the code element:

Without <pre> tag

x = 5; y = 6; z = x + y;

With <pre> tag

    
    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8">
        <title>Example</title>
      </head>
      <body>
        <p>Hello, world!</p>
      </body>
    </html>
