Most Common HTML Block Elements

  • <div> : A generic container for content.
  • <section> : For grouping of content.
  • <h1> - <h6> : HTML headings from most important (<h1>) to least important (<h6>).
  • <p> : Defines a paragraph of text.
  • <header> : To create a header or a set of navigational links.
  • <footer> : Defines a footer for a section or document.
  • <hr> : Creates a thematic break or horizontal rule.
  • <ol> : Represents an ordered (numbered) list.
  • <ul> : Represents an unordered (bulleted) list.
  • <li> : Represents a list item.
  • <main> : Specifies the main content of a document.
  • <nav> : Defines a block of navigation links.
