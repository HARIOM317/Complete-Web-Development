Most Common HTML Inline Elements

  • <span> : A generic inline container for phrasing content.
  • <a> : Defines a hyperlink.
  • <img> : Embeds an image into the document.
  • <small> : Renders text in a smaller size.
  • <b> : Displays bold text without implying extra importance.
  • <i> : Displays italic text.
  • <strong> : Denotes text of strong importance.
  • <em> : Marks text with emphasis, typically rendered in italic.
  • <sub> : Displays subscript text.
  • <sup> : Displays superscript text.
  • <br> : Inserts a line break.
  • <input> : Used for user input in a form.
