Understanding Emmet in VS Code

What is Emmet?

Emmet is a toolkit integrated into Visual Studio Code that helps you write HTML and CSS faster. Instead of typing out long code snippets, you can use simple abbreviations.

How to Use Emmet in VS Code

Example 1: Unordered List with Three Items

Type ul>li*3
  • List item 1
  • List item 2
  • List item 3

Example 2 : Use . for class and # for id attribute

Type .container#content
It is a div with class container and id content

Example 3 : Use elementName.class1.class2.class3 for multiple class 

Type span.class1.class2.class3
It is a span with three classes class1, class2 and class3
